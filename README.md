# Hasmin Arsenal Website

A modern, responsive website for Hasmin Arsenal featuring a clean design and smooth user experience.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Interactive Elements**: Hover effects, animations, and form validation
- **Mobile Menu**: Hamburger menu for mobile navigation
- **Contact Form**: Functional contact form with validation

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Section**: Company information with statistics
3. **Services Section**: Showcase of services offered
4. **Portfolio Section**: Display of projects and work
5. **Contact Section**: Contact information and form
6. **Footer**: Additional links and information

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required - it's a pure HTML/CSS/JavaScript website

## Customization

- **Colors**: Edit CSS variables in `styles.css` (lines 8-15)
- **Content**: Modify text content in `index.html`
- **Images**: Replace placeholder images with actual images
- **Contact Form**: Connect the form to a backend service for actual form submission

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
